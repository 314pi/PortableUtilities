FreeFileSync Portable Issues

Save any configuration files (*.ffs_*) to "App\FreeFileSync" (which should be the default).

To use any of the following features, copy "\FreeFileSyncPortable\Other\Source\FreeFileSyncPortable.ini" to "\FreeFileSyncPortable" then edit the relevant entry.

AdditionalParameters: Batch jobs

  • If you run batch jobs (*.ffs_batch) using AdditionalParameters, make sure the entry points to a file in "App\FreeFileSync" & not "Data\ffs" (these files are not present when the app is running) eg: AdditionalParameters="X:\PortableApps\FreeFileSyncPortable\App\FreeFileSync\job.ffs_batch". Paths with spaces must be quoted.

RealtimeSync: RealtimeSync.exe

  • The RealtimeSync entry allows you to run "RealtimeSync.exe" instead of "FreeFileSync.exe".
  • Values: true / false
  • Default: false
