About Java Portable Launcher
============================
The Java Portable Launcher allows you to quickly run Java JARs using PortableApps.com's Java Portable JVM on any PC.  You can run it directly and select your JAR from a dialog box or pass in the JAR by just dragging and dropping it onto the JavaPortableLauncher.exe icon.
