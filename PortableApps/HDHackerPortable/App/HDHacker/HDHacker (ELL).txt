ΠΕΡΙΓΡΑΦΗ:

Το HDHacker είναι ένα μικρό βοηθητικό πρόγραμμα για Windows, το οποία σας δίνει τη δυνατότητα να αποθηκεύσετε, να προβάλετε ή να αποκαταστήσετε το MBR, τον τομέα εκκίνησης ή οποιοδήποτε άλλο τομέα του σκληρού δίσκου. Το HDHacker μπορεί να κάνει επίσης, αυτή την εργασία για δισκέτα, στικ μνήμης και άλλα μέσα αποθήκευσης. Το HDHacker είναι ΔΩΡΕΑΝ. Το HDHacker δεν χρησιμοποιεί κανένα πρόγραμμα εγκατάστασης, αλλά τρέχει απευθείας από ένα εκτελέσιμο αρχείο.



ΧΡΗΣΗ:

Εκτός από τη μέγιστη εξασφάλιση που παρέχει έναντι επιθέσεων ιών στο MBR (Κύρια Εγγραφή Εκκίνησης), το πρόγραμμα HDHacker έχει γραφτεί ως ένα εργαλείο για μια συνηθισμένη δυσάρεστη θέση:

Σε ένα PC με Linux και Win2000 ή Windows XP (με LILO ή GRUB ως διαχειριστή εκκίνησης) διάφορα λειτουργικά συστήματα μπορεί να αντικαταστήσουν το LILO/GRUB (και να κάνουν το Linux δυσπρόσιτο). Σε αυτή την περίπτωση, το HDHacker σας δίνει τη δυνατότητα να αποθηκεύσετε το MBR πριν από την εγκατάσταση ενός νέου OS (ή την επανεγκατάσταση των Windows), και τότε πολύ εύκολα να επαναφέρετε το αρχικό LILO/GRUB.

Όπως και στις προηγούμενες εκδόσεις, έχετε τη δυνατότητα να επιλέξετε για ανάγνωση/εγγραφή προς/από μια λογική ή φυσική μονάδα δίσκου. Εάν ο πρώτος τομέας έχει επιλεγεί σε μια φυσική μονάδα δίσκου, αυτός θα αντιστοιχεί στο MBR. Διαφορετικά, εάν ο πρώτος τομέας έχει επιλεγεί σε μια λογική μονάδα δίσκου, θα κάνει ανάγνωση/εγγραφή προς/από τον καθορισμένο τομέα εκκίνησης.

Προς το παρόν είναι δυνατόν να επιτευχθούν οποιεσδήποτε από τις ακόλουθες λειτουργίες:

"ΑΝΑΓΝΩΣΗ ΤΟΜΕΑ ΑΠΟ ΔΙΣΚΟ"
Φορτώνει έναν τομέα στη μνήμη, διαβάζοντάς τον απευθείας από το δίσκο.

"ΦΟΡΤΩΣΗ ΤΟΜΕΑ ΑΠΟ ΑΡΧΕΙΟ"
Φορτώνει έναν τομέα στη μνήμη, ο οποίος είχε αποθηκευτεί προηγουμένως σε ένα αρχείο.

"ΑΠΟΘΗΚΕΥΣΗ ΤΟΜΕΑ ΣΕ ΑΡΧΕΙΟ"
Αποθηκεύει τον τομέα που βρίσκεται στη μνήμη, σε ένα αρχείο σε κάποιο καθορισμένο δίσκο ή μέσο.

"ΕΓΓΡΑΦΗ ΤΟΜΕΑ ΣΕ ΔΙΣΚΟ"
Εγγράφει στο δίσκο, τον τομέα που βρίσκεται στη μνήμη, στην καθορισμένη θέση (MBR ή τομέα εκκίνησης).

Στη συνέχεια, για να αποθηκεύσει τα δεδομένα (επιλεγμένο MBR ή τομέας εκκίνησης), το HDHacker θα ζητήσει:

1. Να επιλέξετε μια φυσική μονάδα δίσκου.
2. Να επιλέξετε "ΑΝΑΓΝΩΣΗ ΤΟΜΕΑ ΑΠΟ ΔΙΣΚΟ"
3. Να επιλέξετε "ΑΠΟΘΗΚΕΥΣΗ ΤΟΜΕΑ ΣΕ ΑΡΧΕΙΟ"
   (Εάν θέλετε να αποθηκεύσετε το MBR σε ένα αρχείο, π.χ. ως αντίγραφο ασφαλείας).

Κατά τη διάρκεια της ΕΠΑΝΑΦΟΡΑΣ, οι επιλογές θα είναι:

1. Επιλογή μιας φυσικής μονάδας δίσκου.
2. Επιλογή "ΦΟΡΤΩΣΗ ΤΟΜΕΑ ΑΠΟ ΑΡΧΕΙΟ" 
   (στη συνέχεια επιλογή του αρχείου ασφαλείας MBR ή τομέα που έχετε αποθηκεύσει προηγούμενα).
3. Επιλογή "ΕΓΓΡΑΦΗ ΤΟΜΕΑ ΣΕ ΔΙΣΚΟ" για να ολοκληρωθεί η διαδικασία.



ΣΥΜΒΑΤΟΤΗΤΑ / ΑΠΑΙΤΗΣΕΙΣ:

Θεωρητικά το HDHacker πρέπει να δουλεύει σε όλες τις εκδόσεις των Windows, με τους ακόλουθους περιορισμούς:

ΔΙΚΑΙΩΜΑΤΑ: Το πρόγραμμα απαιτεί ο χρήστης να έχει δικαίωμα ανάγνωσης ΚΑΙ εγγραφής σε ΟΠΟΙΟΔΗΠΟΤΕ τμήμα του δίσκου. Στα Windows 95 και 98, αυτή είναι η προεπιλογή εάν έχετε κάνει σύνδεση με έναν κωδικό πρόσβασης (αλλιώς πρέπει να δημιουργήσετε έναν λογαριασμό διαχειριστή). Ωστόσο στα Windows ME, ή οποιαδήποτε συστήματα βασισμένα σε ΝΤ (Windows 2K, Windows XP κλπ) θα πρέπει να συνδεθείτε ως "Διαχειριστής" για να διαβάσετε, να αποθηκεύσετε ή να επαναφέρετε οποιοδήποτε MBR ή τομέα εκκίνησης.

LINUX: Το πρόγραμμα HDHacker λειτουργεί μέσα σε Linux, χρησιμοποιώντας το WINE, αλλά πάλι θα πρέπει να είστε "Διαχειριστής" και θα πρέπει επίσης να έχετε δικαιώματα ανάγνωσης/εγγραφής στο FSTAB.

Το HDHacker απαιτεί το MSVBVM60.DLL για να τρέξει σε Windows 95 ή Windows 98.
(Συμβουλή: Το MSVBVM60.DLL είναι ένα τυπικό συστατικό αρχείο στα Windows 2000)
